pylade.allowed_classes module¶

Define classes that can be used as corpus readers and implementations.

Classes need to be whitelisted in order to sanitize command-line input and avoid malicious or wrong code execution.

Utility functions shared across the project.

convert_unknown_arguments(dictionary)[source]¶

Convert strings in dictionary to their proper typed values.

Digits are automatically evaluated by json. We need to evaluate booleans.

Parameters:dictionary (dict) – The dictionary whose values need to be converted. 
>>> ex_dict = { 'flag': 'true', 'verbose': 'False' }
>>> convert_unknown_arguments(ex_dict) == {'flag': True, 'verbose': False}
True

nested_defaultdict()[source]¶

Allow to avoid Pickle errors with lambdas.

This function is defined in order to avoid Pickle errors. E.g.:

>>> AttributeError: Can't pickle local object 
... 'TwitterCorpusReader.ngram_frequencies.<locals>.<lambda>'
Returns:a defaultdict of integers.

Note

Functions are pickled by name, not by code. Unpickling will only work if a function with the same name is present in in the same module. This is why pickling a lambda won’t work: they have no individual names.
